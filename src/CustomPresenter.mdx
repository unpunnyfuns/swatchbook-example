# Custom color presenters

The built-in blocks render each token `$type` through a [presenter](https://unpunnyfuns.github.io/swatchbook/guides/creating-custom-presenters/), and you can supply your own. Wrap a region in `PresenterContext.Provider` with a merged registry to override how a type renders inside it, while the rest of your docs stay on the built-in (see `Tokens/Overview`).

## A bespoke swatch

`DualSurfaceSwatch` (`src/DualSurfaceSwatch.tsx`) paints each colour over both a light and a dark surface, so you can judge it against either background:

<PresenterContext.Provider value={mergePresenters({ color: DualSurfaceSwatch })}>
  <ColorPalette filter="color.**" />
</PresenterContext.Provider>

## Applying one everywhere

To apply a presenter across the whole Storybook rather than one subtree, pass it to the [addon](https://unpunnyfuns.github.io/swatchbook/reference/addon/) in `.storybook/preview.ts`:

```ts
swatchbookAddon({ presenters: { color: DualSurfaceSwatch } })
```

That registration reaches every block, in both stories and MDX docs pages.
